# Book-Store-MERN-Stack

### Backend
```bash
    npm init -y
    npm i express nodemon
    npm run dev
    npm i mongoose
    npm i cors
```

### Fronteend
```bash
    npx create-react-app frontend
    cd frontend
    npm install axios react-router-dom
    npm start
    npm i axios react-icons
    
    // for Display notifications with call of a function
    npm install notistack
```
